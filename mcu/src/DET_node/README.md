# ESP8266键盘接线指南

## 4x4矩阵键盘连接说明

### 重要纠正
⚠️ 当前接法错误！矩阵键盘必须区分行和列引脚，不能将同一行的按键接同一引脚。

### 正确引脚定义
- 行引脚 (ROW1-ROW4): 连接键盘的4个行线，接ESP8266的D0-D3
- 列引脚 (COL1-COL4): 连接键盘的4个列线，接ESP8266的D5-D8

### 矩阵键盘原理
键盘采用行列扫描方式，必须按照以下物理连接：
- 每行按键共享一个行引脚
- 每列按键共享一个列引脚

### 正确接线示例

```
键盘物理布局：
行1: 1, 2, 3, A → 接D0
行2: 4, 5, 6, B → 接D1  
行3: 7, 8, 9, C → 接D2
行4: *, 0, #, D → 接D3

列1: 1,4,7,* → 接D5
列2: 2,5,8,0 → 接D6
列3: 3,6,9,# → 接D7
列4: A,B,C,D → 接D8
```

### 键盘布局

```
+-----+-----+-----+-----+
|  1  |  2  |  3  |  A  |
+-----+-----+-----+-----+
|  4  |  5  |  6  |  B  |
+-----+-----+-----+-----+
|  7  |  8  |  9  |  C  |
+-----+-----+-----+-----+
|  *  |  0  |  #  |  D  |
+-----+-----+-----+-----+
```

### 注意事项
1. 确保接线牢固，避免接触不良
2. 上电前检查接线是否正确
3. D3(GPIO0)引脚在启动时需要保持高电平
4. 建议使用杜邦线连接，便于调试

### 测试方法
1. 上传程序到ESP8266
2. 打开串口监视器(115200波特率)
3. 按下键盘按键，观察串口输出
